### The two examples here are the Spring Cloud microservice project using Nacos as the registry and the transformation of it to Spring Cloud Huawei microservice project using CSE as the registry

## nacos-springcloud Project:
   Based on the Spring Cloud framework, Nacos is a simple microservice structure for the registry

## nacos-springcloud-huawei Project:
   Based on the Spring Cloud Huawei framework, the CSE is a simple microservice structure for the registry
